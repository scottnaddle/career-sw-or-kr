신청 절차

SW기술자 경력등록을 위한 5단계 신청 절차

1

회원가입

개인/기업 회원가입 및 본인인증

  • 온라인 회원가입
  • 휴대폰 본인인증
  • 이메일 인증
  • 개인정보 수집 동의
2

신청서 작성

경력 정보 입력 및 신청서 작성

  • 기본 정보 입력
  • 학력/자격 정보
  • 경력 상세 내용
  • 업무 수행 내역
3

서류 제출

증빙서류 업로드 및 제출

  • 신분증 사본
  • 학력/자격증 사본
  • 경력증명서
  • 업무수행확인서
4

경력 신청

최종 신청 및 수수료 결제

  • 신청 내용 최종 확인
  • 수수료 결제
  • 신청서 제출
  • 접수번호 발급
5

완료/확인

심사 완료 및 결과 확인

  • 서류 심사 (3-5일)
  • 결과 통보
  • 경력 등록 완료
  • 확인서 발급 가능

💡 신청 시 유의사항

📝
정확한 정보 입력

모든 정보는 정확하게 입력해주세요. 허위 정보 입력 시 제재를 받을 수 있습니다.

📋
서류 준비

필요한 서류를 미리 준비하여 신청 과정을 원활하게 진행하세요.

⏰
처리 기간

서류 심사에는 평균 3-5일이 소요되며, 추가 서류 요청 시 기간이 연장될 수 있습니다.

💰
수수료 결제

신청 시 결제한 수수료는 심사 결과와 관계없이 환불되지 않습니다.

📄 필수 제출 서류

📎 추가 제출 서류

업무 수행 관련
  • 프로젝트 수행확인서 (회사 발급)
  • 업무기술서 (본인 작성)
  • 포트폴리오 (선택사항)
  • 고객 추천서 (프리랜서인 경우)
교육 이수 관련
  • 교육과정 수료증
  • 온라인 강의 이수증
  • 세미나/컨퍼런스 참석증
  • 기술 교육 수료증
기타 증빙 서류
  • 소득금액증명원 (개인사업자)
  • 부가가치세 과세표준증명원
  • 계약서 사본 (프로젝트별)
  • 작업물 증빙 자료

📋 서류 제출 가이드

🔍 서류 품질 기준
  • 해상도: 300dpi 이상, 선명하게 읽을 수 있는 상태
  • 파일 형식: PDF, JPG, PNG 형식만 업로드 가능
  • 파일 크기: 개별 파일당 최대 10MB
  • 컬러: 컬러 스캔 권장 (흑백도 가능)
📝 서류 작성 요령
  • 업무기술서: 구체적인 업무 내용과 사용 기술 명시
  • 프로젝트명: 실제 수행한 프로젝트명과 기간 정확히 기재
  • 담당 업무: 본인이 수행한 구체적인 역할과 책임 명시
  • 기술 스택: 사용한 프로그래밍 언어, 도구, 프레임워크 기재
⚠️ 주의 사항
  • 위조 방지: 모든 서류는 원본 또는 공증된 사본만 인정
  • 개인정보: 불필요한 개인정보는 마스킹 처리 가능
  • 유효기간: 발급일로부터 3개월 이내 서류만 유효
  • 번역본: 외국 서류는 공증된 한글 번역본 첨부 필요

기업회원 서비스

기업을 위한 전용 서비스와 대리 신청 절차 안내

🏢 기업회원 서비스 개요

기업회원은 소속 직원의 SW기술자 경력등록을 대리 신청하고, 기업 인증서를 활용한 전자서명으로 보다 신속하고 안전하게 업무를 처리할 수 있습니다.

✨ 기업회원 혜택

🚀
신속한 처리
  • 우선 심사 서비스
  • 대량 신청 시 일괄 처리
  • 전담 담당자 배정
💰
비용 절감
  • 대량 신청 할인 혜택
  • 연간 계약 시 추가 할인
  • 수수료 후불 결제 가능
📊
통합 관리
  • 직원 경력 현황 통합 조회
  • 부서별/프로젝트별 관리
  • 통계 및 분석 리포트 제공
🔒
보안 강화
  • 기업 공인인증서 연동
  • 전자서명 자동 처리
  • 보안 통신 채널 제공

구분 서비스 수수료 처리기간 비고

추가 경력 등록 15,000원 3-5일 건당
연간 관리 경력 정보 관리 5,000원 - 매년 갱신
확인서 발급 경력확인서 (기본) 2,000원 즉시 첫 1매 무료
경력확인서 (영문) 5,000원 1-2일 해외 제출용
상세 이력서 3,000원 즉시 프로젝트 상세 포함
정보 변경 기본 정보 수정 무료 즉시 년 3회까지
경력 내용 수정 10,000원 1-3일 재심사 필요

📋 수수료 정책

환불 정책
  • 신청 취소: 서류 심사 전까지 100% 환불
  • 심사 중 취소: 50% 환불 (행정비용 차감)
  • 심사 완료 후: 환불 불가 (승인/반려 무관)
  • 환불 처리: 신청일로부터 7일 이내
할인 혜택
  • 신규 회원: 첫 등록 시 20% 할인
  • 재등록: 기존 회원 10% 할인
  • 다량 신청: 10건 이상 시 단계별 할인
  • 학생 할인: 졸업 후 6개월 이내 50% 할인
유의 사항
  • 가격 변동: 사전 공지 후 변경 가능
  • 부가세: 모든 수수료는 부가세 별도
  • 영수증: 결제 완료 시 자동 발행
  • 문의: 수수료 관련 문의 시 고객센터 연락
